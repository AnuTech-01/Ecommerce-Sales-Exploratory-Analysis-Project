## 📐 DAX Measures Created

```dax
Total Sales = SUM('Ecommerce Dataset'[Gross Amount])

Avg Sales = AVERAGE('Ecommerce Dataset'[Gross Amount])

Num Customers = DISTINCTCOUNT('Ecommerce Dataset'[CID])

Num Transaction = DISTINCTCOUNT('Ecommerce Dataset'[TID])
```

## 🧹 Data Cleaning Steps
